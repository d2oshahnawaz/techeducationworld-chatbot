# 🤖 Tech Education World Chatbot

An AI-powered chatbot designed for **Tech Education World** to assist students with queries, learning resources, and guidance.

## 🚀 Features

* 💬 Interactive Chat Interface (Tkinter GUI)
* 🧠 AI-based Intent Recognition
* 📚 Provides educational assistance
* ⚡ Fast and user-friendly responses
* 🎯 Designed for students and learners

## 🛠️ Technologies Used

* Python 🐍
* Tkinter (GUI)
* Machine Learning / NLP (Intent Prediction)

## 📂 Project Structure

```
├── chatbot_model.py
├── train_model.py
├── intents.json
├── main.py
└── README.md
```

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/techeducationworld-chatbot.git
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the chatbot:

```
python main.py
```

## 🎯 Purpose

This chatbot is built to support the mission of **Tech Education World** by providing free and accessible learning assistance to students.
